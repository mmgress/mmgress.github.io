# Auralion PVT test page

Statyczna strona testowa do publikacji na GitHub Pages.

## Pliki

- `index.html` - struktura strony i tresc landing page.
- `styles.css` - pelny styl responsywny.
- `script.js` - interaktywny wybor kohort, formularz diagnostyczny i nawigacja mobilna.

## Uruchomienie

Wystarczy otworzyc `index.html` w przegladarce albo opublikowac katalog jako GitHub Pages.

Strona jest prototypem komunikacji. Parametry techniczne, oszczednosci i dobor systemu wymagaja audytu konkretnego obiektu.
